## Project Setup Instructions

1. **Database Setup**:
   - Open MySQL Workbench and run `database_setup.sql` located in the `sql/` folder.
   - This will create the required database and tables.

2. **Python Setup**:
   - Make sure you have Python installed.
   - Install `mysql-connector-python` by running:
     ```bash
     pip install mysql-connector-python
     ```
   - Run the Python file in `python_code/studentfile.py`.

3. **Running the Project**:
   - After setting up the database, you can run the project by executing the code in `python_code/studentfile.py`.
